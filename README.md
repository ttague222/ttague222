## Featured Work

**[RailWatch: Payment Operations Monitor](https://github.com/ttague222/Railwatch-Payment-Ops-Monitor)**
A self-directed fintech build: a real-time operations dashboard for community-bank payments teams. One
unified view across ACH, wire, RTP, and FedNow rails, with exception queues, settlement tracking, and
live market context. Built spec-first with a clean DataProvider architecture ready for core-banking
integration. Live demo: [railwatch-payment-ops-monitor.vercel.app](https://railwatch-payment-ops-monitor.vercel.app).
`React · TypeScript · Vite · Vercel · REST APIs`
